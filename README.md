# Swastik Media Solutions

Premium SaaS website designed with Antigravity Aesthetics and Glassmorphism 2.0.

## 🚀 Features
- 🌫️ Multi-layered Glassmorphism with 40px backdrop blur.
- 🐚 Golden Ratio typography (1.618 scale).
- 🌊 Animated abstract wave backgrounds.
- ⚡ High-performance Vite environment.
- 📱 Fully responsive grid-based layouts.

## 🛠️ Stack
- HTML5 / CSS3 (Vanilla)
- JavaScript (ES6+)
- Vite
- Inter Typeface

## 📦 Getting Started
1. Clone the repo
2. Run `npm install`
3. Start the dev server with `npm run dev`
